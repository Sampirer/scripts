# Scripts

Sammlung nützlicher Shell-Scripts für Arch Linux.

## Installation

```bash
# Repository klonen
git clone https://github.com/Sampirer/scripts ~/repos/scripts
cd ~/repos/scripts

# Scripts verfügbar machen
stow -t ~ .
```
